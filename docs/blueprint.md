# **App Name**: MathMentor

## Core Features:

- Lesson Management: Teachers can securely log in to create, edit, and organize math lessons using a rich text editor with LateX support and embed media like videos and images.
- Interactive Lesson Viewer: Students can navigate lessons, view content, and engage with embedded interactive elements on any device.
- Practice Questions & Feedback: Integrate short, formative assessment questions (e.g., multiple-choice, fill-in-the-blank) within lessons, providing immediate feedback to students.
- Student Progress Dashboard: Students can view their progress, including which lessons they've completed and their performance on practice questions.
- AI-powered Explainer Tool: Students can use a generative AI tool to ask for further clarification or alternative explanations for difficult math concepts directly within a lesson.
- User Authentication: Secure login and registration system for both teachers and students, maintaining separate dashboards and access levels.

## Style Guidelines:

- Primary color: A balanced, professional blue (#3986AC). This hue was chosen to evoke trust, clarity, and focus, suitable for an educational context.
- Background color: A very light, desaturated blue (#EBF3F5). Its subtle hue aligns with the primary color, creating a calm and clean canvas that enhances readability for long study sessions.
- Accent color: A vibrant, clear aqua-green (#5BE3CC). Analogous to the primary color, this bright, yet professional, hue is perfect for highlighting interactive elements and calls to action without being overwhelming.
- Font choice: 'Inter' (sans-serif) for all text elements, including headlines and body. This font offers excellent legibility and a modern, neutral aesthetic, ensuring a clear and focused reading experience crucial for educational content.
- Use a consistent set of clean, outline-style icons to represent educational tools, progress markers, and interactive elements. Simplicity and clarity will be key.
- Implement a structured, responsive layout with ample white space. Content will be presented in a clear hierarchy to guide students through lessons effortlessly on any device.
- Animations will be subtle and purposeful, primarily used for feedback on interactive quizzes (e.g., correct/incorrect answer indications) and smooth transitions when navigating between lesson sections.
